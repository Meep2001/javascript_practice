The event the is generated on the p-element i.e the click event bubbles and reaches the parent element. To stop bubbling use event.stopPropogation This will stop the event to propogate to the parent element, incase the element where the handler has multiple event listeners those event listeners will be executed i.e element.addEventListener('click',(e)=>{alert(1); e.stopPropogation()}) element.addEventListener('click',()=>alert(2)) Here the bubbling will stop but the calling of the second handler wont stop. At any point the event.target will give the element that triggered the event whereas the 'this' keyword will give the element whose handler is currently handling the event. Therefore this === event.target may be true sometimes
